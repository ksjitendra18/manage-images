# About Project

In this project users can upload the image to their BunnyCdn account and can get the link copied of the uploaded images.

Through this project I am learning Vue. I will highly appreaciate any feedbacks.
## Tech Stack
- **UI**: Vue.js
- **Styling**: TailwindCSS, DaisyUI
- **Backend**: Netlify Functions
- **Database**: Planetscale
- **ORM**: Drizzle
- **Hosting**: Netlify
